2. Features

The Exception Reporter display is grouped by 6 tab pages. These tab pages are General, Exceptions, Assemblies, Settings, Environment and Contact.

The user is presented with 4 control buttons labelled Print, Save, Copy, E-Mail.

The Exception Reporter's functionality is described in terms of the content provided on these tab pages and the actions performed by pressing the control buttons. This information can be found by navigating the links on the index to the left of this page.
