3. Properties

The tab pages displayed, control buttons available and a number of messages are customizable by configuring the properties of the specific instance of the Exception Reporter used by the application. Properties of the Exception Reporter are:

Property Type Description
ShowGeneralTab Boolean If true the General Tab Page is shown, if false it is hidden.
ShowExceptionsTab Boolean If true the Exceptions Tab Page is shown, if false it is hidden.
ShowAssembliesTab Boolean If true the Assemblies Tab Page is shown, if false it is hidden.
ShowSettingsTab Boolean If true the Settings Tab Page is shown, if false it is hidden.
ShowEnvironmentTab Boolean If true the Environment Tab Page is shown, if false it is hidden.
ShowContactTab Boolean If true the Contact Tab Page is shown, if false it is hidden.
 
ShowPrintButton Boolean If true the Print control button is shown, if false it is hidden.
ShowSaveButton Boolean If true the Save control button is shown, if false it is hidden.
ShowCopyButton Boolean If true the Copy control button is shown, if false it is hidden.
ShowEmailButton Boolean If true the E-mail control button is shown, if false it is hidden.
 
ContactEmail String The E-mail address shown on the Contact tab page. If the SendEmailAddress is blank, this property is also the address that e-mails are sent to on press of the E-mail button.
ContactWeb String The Web address shown on the Contact tab page.
ContactPhone String The Phone number shown on the Contact tab page.
ContactFax String The Fax number shown on the Contact tab page.
 
MailType Enumerated SMTP or SimpleMAPI. If SMTP then e-mail is sent through a specified SMTP server, if SimpleMAPI then the user's mail client is used.
SendEmailAddress String The E-mail address that mail is sent to on click of the E-mail control button. If this property is blank, then the address specified by the ContactEmail property is used.
 
SMTPServer String The SMTP server to send e-mail through. If this property is blank, then the local machine is used.
SMTPFromAddress String The address that SMTP e-mail should appear to be sent from.
SMTPUsername String If specified, this username is sent to the SMTP server for authentication (.Net framework 1.1 and higher only).
SMTPPassword String If specified, this password is sent to the SMTP server for authentication (.Net framework 1.1 and higher only).
 
General Message String The General Message displayed at the top of the General tab page.
Explanation Message String The Message displayed above the box intended for the user to type information relating to the actions they took imediately prior to the error occurring.
ContactMessageTop String Message displayed at the top of the Contact tab page.
ContactMessageTop String Message displayed at the bottom of the Contact tab page.
EnumeratePrinters Boolean If true, the list of installed printers is included in the ExceptionReporter display. NOTE: The printer enumaration process has been known to hang on some machines (this seems dependant on the printer drivers installed), to avoid this issue set this value to false
