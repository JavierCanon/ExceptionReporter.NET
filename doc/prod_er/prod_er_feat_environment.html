2.5 Environment

The Environment tab displays information about:
  • the user's logged on username
  • the name of the machine on which the error occurred
  • system processor
  • memory
  • operating system
  • drives
  • environment variables
  • printers
  • other system information

Errors sometimes occur in one environment and not another, this information is relevant where it is suspected that the error be somehow related to the environment the application is running within.
