2.1 General

The General tab is the opening tab displayed to the user on application error. This tab page provides the user with:
  • a customisable message
  • the time and date the error occurred
  • the name and version of the running application
  • the region of the running process
  • a box for the user to enter information about the actions they were taking prior to the error.
This tab page displays non-technical information, and simply informs the user that a problem has occurred, showing some basic application information. The user is requested to provide information describing actions taken leading up to the error, this type of information is valuable when trying to replicate the error in order to find a solution.
